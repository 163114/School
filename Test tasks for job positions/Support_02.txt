Вводная информация:
Вы - член команды тестирования прибора сбора данных торгового зала. Ваша задача составить план тестирования данного прибора, чтобы определить корректность его работы. Данный прибор используется для распознавания товаров, находящихся в зале, путем сканирования штрих-кодов. Также используется для сравнения с существующими товарами в базе данных. При сканировании на экране прибора выводятся следующие параметры:
●	код товара – 16-значное числовое поле;
●	наименование товара;
●	страна-производитель – изображение флага страны производителя;
●	время поступления на склад;
●	количество на складе.
Задание:
1. указать, какие, на ваш взгляд, у данного прибора должны быть функциональные кнопки;
2. написать порядок ваших действий для проверки функциональности и корректности
выведения информации на данном приборе;
3. указать ваши предложения по модификации и улучшению данного прибора.
